## 📌 Sobre o Projeto
Este projeto realiza a análise exploratória de dados abertos da **ANEEL** (Agência Nacional de Energia Elétrica), por meio do Sistema SIGET — Sistema de Gestão da Transmissão.

A base contém informações sobre obras, contratos, situação de empreendimentos, receitas e tipos de instalações do setor elétrico brasileiro.

**Base de dados original:** https://dados.gov.br/dados/conjuntos-dados/sistema-de-gestao-da-transmissao-siget

---

## ❓ Perguntas respondidas
1. Quantos empreendimentos estão em cada situação: *Planejado, Em Operação, Concluído*?
2. Quantas obras são de cada tipo: *Instalação, Adequação, Substituição, Ampliação, etc.*?
3. Quantidade de contratos finais previstos por ano?
4. Quantas obras foram solicitadas por cada empreendimento?
5. Quantidade de cada tipo de módulo presente nas obras?
6. Quantos contratos existem por identificador único?
7. Qual o valor da receita de uma determinada empresa/empreendimento?
8. Qual a data de previsão de conclusão de uma obra específica?
9. Qual obra apresenta o maior valor de receita do último ciclo tarifário?
10. Quais são os 4 tipos de instalações mais comuns?
